EVENT 
- 브라우저를 통해서 특정 웹페이지 도착
- 사용자가 하는 행위 모든 것
- 페이지를 확인 / 버튼 클릭 / 스크롤 이동
- 이벤트를 설정
- event Hander => onClick
- addEventListener
- 이벤트 설정 > 로컬 컴퓨터 > 메모리 공간 > 이벤트 객체 생성 > 이벤트 객체를 참조변수 값으로 찾아올 수 있다
- 이벤트 객체 > preventdefault유일한 메서드 함수


이벤트 속성 종류

1. 문서(window) 로딩 이벤트
 load


2. 마우스 이벤트 
- click : 사용자해 해당 element를 클릭했을 때(버튼을 눌렀다가 떼었을 때) 발생 합니다.
- dblclick  : 사용자가 해당 element에서 마우스 버튼을 더블 클릭했을 때 발생합니다
- mouseover : 특정 요소 위에 마우스를 올렸을 떄 (버블링 기본적용)
- mouseout  : 특정 요소 위에 마우스를 떠나게 했을 떄 (버블링 기본적용)
- mouseenter : 특정 요소 위에 마우스를 올렸을 떄 (버블링 X)
- mouseleave : 특정 요소 위에 마우스를 떠나게 했을 떄 (버블링 X)

( * 버블링 :  마우스가 올라간 해당 요소만 작동 되는거 => 버블링 X


3. 키보드 이벤트
- keydown : 사용자가 키를 누르고 있는 동안 발생되는 이벤트.
- keyup : 사용자가 키를 눌렀다가 떼었을 때 발생하는 이벤트.

5. 위치 이벤트
 위치를 기준으로 특정 이벤트를 발생시키는 이벤트

- page X : 문서 (코드를 작성한 작업데이터)를 기준으로 x축으로 부터 얼만큼 이동한 곳인지 확인
- page Y : 문서 (코드를 작성한 작업데이터)를 기준으로 y축으로 부터 얼만큼 이동한 곳인지 확인
- screen X : 현재 화면을 기준으로 x축으로 부터 얼만큼 이동한 곳인지 확인
- screen Y : 현재 화면을 기준으로 y축으로 부터 얼만큼 이동한 곳인지 확인
